The Architecture

To support real-time editorial demands, the platform was built using Wordpress, Elementor, and JSP. The development centered on merging engineering best practices with clean user experience, resulting in a robust and highly intuitive environment for journalists.

"Fusing strict engineering logic with intuitive design to deliver digital platforms that scale without friction."

The Execution

I led a four-person development team under SK Solutions, acting as the primary project manager. By maintaining strict project timelines and emphasizing team coordination, we launched a high-performance site capable of handling heavy, continuous content updates.
