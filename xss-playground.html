The Project Story

This lab was engineered to provide a visual bridge between offensive payloads and defensive coding. It focuses on DOM-based XSS, the most common vector in modern JS-heavy applications.

Architecture

The lab uses a vanilla JS engine that toggles between innerHTML and textContent nodes. This side-by-side simulation proves that security is often just a matter of choosing the correct API for data rendering.

The Trenches

Managing the scope of the payloads was a significant challenge. To prevent an injected script from compromising the entire portfolio's performance, I implemented strict containment rules within the Victim View's DOM subtree.
