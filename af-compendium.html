NGO Resource Hub

AF Compendium

Scale: 267 Organizations Stack: Custom Search Engine Visit Live Site

A centralized digital directory developed for the Association of Foundations to facilitate cross-sector collaboration.

The Architecture

Built as a robust resource microsite, the core functionality relies on a custom-coded keyword search matching system. It allows hundreds of NGOs to find and connect with each other based on specific developmental sectors and geographic locations.

The Trenches

Managing data for 267 organizations required a strict mapping protocol. I developed a logic that ensures high-accuracy search results even with partial keyword entries, drastically reducing the friction in NGO communication.

The Study

The UX & Information Architecture Strategy

To accommodate a non-technical user base, we drastically simplified the interface by implementing a strict "3-clicks" rule, ensuring any piece of information across the massive compendium is accessible within exactly three clicks. We abandoned complex, multi-level menus in favor of a single, unified main-tier layout to keep users anchored and completely eliminate navigational friction.

Visually, we designed the platform to feel less like a traditional website and more like an authoritative encyclopedia or clean database. By pairing this highly structured directory look with generous spacing and strong typographic hierarchies, we minimized visual noise, making massive volumes of data highly scannable, organized, and immediately approachable.

The UI & Technical Solution

To make this massive database scalable, we leveraged dynamic WordPress Posts on the backend. This allowed us to systematically manage all 267 organizations while guaranteeing each profile automatically inherited the exact same structured layout.

At the core of the UI is a frictionless instant search engine. Because non-technical users are often intimidated by complex filters, our search populates results immediately as they type—no page reloads required. We paired this speed with extremely clear, action-oriented UI components. Every directory listing features a short, explicit description tied directly to a high-visibility Call-to-Action (CTA) button, completely removing guesswork and giving users absolute clarity on their next step.
