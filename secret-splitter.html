MECHANICAL RECONSTRUCTION
THE STORY

Visual Cryptography

This project deconstructs the complexity of encryption into a physical act. By splitting a message into two noise patterns, we achieve 'Perfect Secrecy'—a state where no amount of brute-force computing can reveal the secret if only one share is present.

ARCHITECTURE

Pixel XOR Manipulation

Every pixel in Share A is a random boolean (Black/White). For Share B, if the original message pixel is 'On', the pixel is the inverse of Share A. If 'Off', it matches Share A. The stacking of these patterns creates the visual contrast necessary for decryption via the human eye.
