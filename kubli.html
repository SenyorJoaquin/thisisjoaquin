Cryptography & Steganography

Kubli

Award: Inabel - Best Solution Stack: Python GitHub Repository

A sophisticated steganography tool that allows users to hide encrypted text within image files.

The Architecture

Built using Python, Kubli utilizes Least Significant Bit (LSB) steganography. It manipulates the binary data of pixels to store hidden messages that are visually undetectable to the human eye, coupled with a secondary encryption layer for maximum privacy.

The Trenches

Maintaining image integrity while injecting data was the core hurdle. I optimized the bit-shifting logic to ensure that even large blocks of text do not produce visible artifacts or noise in the carrier image.
